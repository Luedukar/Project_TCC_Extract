# Projeto TCC

Esse repositorio é dedicado ao versionamento do projeto de TCC para o curso de engenharia de software

Atualmente o repositório conta com:

- pipeline para extração de preços, normalização de valores e resposta ao cliente final

Tecnologias utilizadas até o momento:

- Python (bibliotecas mais relevantes: psycopg2, smtplib, selenium, BeautifulSoup e schedule)
